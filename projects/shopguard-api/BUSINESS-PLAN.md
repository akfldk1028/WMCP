# ShopGuard Business Plan

## 핵심 전략: B2C 초저가 + 크롬 확장

> 크롬 확장이 메인 제품. API는 확장의 백엔드.
> 엄청 싸게 해서 사용자 수로 승부.

## 시장 기회

### Fakespot 종료 = 17M MAU 빈 자리
- Mozilla가 인수한 Fakespot: **2025.7.1 서비스 완전 종료**
- 17M 월간 사용자가 대안을 찾고 있음
- 기존 경쟁: 대부분 죽었거나 엔터프라이즈 전용

### 규제 태풍
- EU Digital Fairness Act: 2026 Q4 발의
- GDPR 다크패턴 벌금 누적 €16억+
- 미국 FTC + 3개 주법: 다크패턴 동의 무효

## 제품: 크롬 확장 (메인)

### 무료 기능
- 다크패턴 감지 (9종): urgency, scarcity, hidden costs, ...
- 페이지 신뢰도 점수 (Trust Score)
- 기본 가격 분석

### Pro 기능 ($2.99/월 또는 $24.99/년)
- 가짜 리뷰 감지 (7개 신호 분석)
- 숨겨진 수수료 상세 분석
- 가격 히스토리 추적
- 안전한 대안 상품 추천
- 리뷰 요약 (AI)

### 왜 $2.99인가?
- Fakespot = 무료였지만 서비스 종료됨
- 사용자는 "약간의 돈"은 낼 의향 있음 (커피 한잔 이하)
- $49/월은 개인한테 비쌈 → $2.99면 충동구매 영역
- 1만 유료 사용자 = **월 $30K** (연 $360K)
- 10만 유료 사용자 = **월 $300K** (연 $3.6M)

## 가격 전략: B2C 싸게, B2B 비싸게

### B2C (크롬 확장 — 개인 사용자)

| 티어 | 가격 | 기능 |
|------|------|------|
| Free | $0 | 다크패턴 9종 감지, Trust Score, 기본 가격 경고 |
| Pro | **$2.99/월** ($24.99/년) | + 가짜 리뷰 7신호, 숨겨진 수수료 상세, 가격 추적 |

→ 커피 한잔보다 쌈. 충동구매 영역.

### B2B API (기업/플랫폼)

| 티어 | 가격 | 기능 |
|------|------|------|
| Starter | **$99/월** | 5K req/day, 다크패턴+리뷰+가격 전체 API |
| Business | **$299/월** | 50K req/day + 정기 사이트 스캔 + PDF 컴플라이언스 보고서 |
| Enterprise | **$999+/월** | 무제한 + 전담 지원 + 커스텀 룰 + SLA |

→ B2B에만 주는 좋은 기능:
- **컴플라이언스 보고서** (EU DSA/DFA 대응, PDF 자동 생성)
- **정기 스캔** (매주/매일 사이트 크롤링 → 위반 알림)
- **대시보드** (다크패턴 트렌드, 리뷰 건전성 모니터링)
- **Webhook 알림** (새 위반 발견시 Slack/Email)
- **벌크 분석** (수백 페이지 일괄 스캔)
- **커스텀 다크패턴 룰** (업종별 맞춤 규칙)
- **감사 로그** (규제 기관 제출용)

### 수익 구조

| 채널 | 예상 비중 | 설명 |
|------|----------|------|
| B2B API ($99~$999/월) | **45%** | 기업 컴플라이언스 (고단가, 안정적) |
| B2C Pro ($2.99/월) | 25% | 크롬 확장 프리미엄 (대량, 저단가) |
| 어필리에이트 | 25% | 안전한 대안 상품 추천 |
| 데이터 리포트 | 5% | 다크패턴 실태 보고서 판매 |

### 왜 이 구조가 작동하는가
- **B2C 싸게** → 사용자 수 폭발 → 시장 지배력 + 어필리에이트 수익
- **B2B 비싸게** → 기업은 벌금 회피가 목적이므로 $299도 싸게 느낌 (벌금 €746M vs $299/월)
- **좋은 기능은 B2B에만** → 보고서, 정기 스캔, 대시보드 = 기업만 필요
- B2C 무료 사용자도 어필리에이트로 수익 발생 (전환 안 해도 돈 됨)

### 어필리에이트 상세
- "이 상품에 가짜 리뷰가 많습니다 → 대안 추천:"
- Amazon Associates: 1~10% 수수료
- Honey 모델: 17M MAU → PayPal $40억 인수
- 쿠팡 파트너스: 3% 수수료 (한국 시장)

## 기술 아키텍처

```
[크롬 확장] ──→ [ShopGuard API (Vercel)]
   │                    │
   │ 무료: 클라이언트    │ Pro: 서버 분석
   │ 사이드 분석        │ (shopguard-mcp)
   │                    │
   ├─ 다크패턴 (로컬)   ├─ 리뷰 7신호
   ├─ Trust Score       ├─ 숨겨진 수수료
   └─ 기본 가격         └─ AI 요약
```

### 무료 = 로컬 분석 (API 비용 0)
- darkpattern-signals.ts → 크롬 확장 번들에 포함
- 서버 호출 없이 HTML 직접 분석
- API 비용 없으므로 무제한 무료 가능

### Pro = API 호출
- 크롬 확장에서 API 호출 → 서버가 분석
- rate limit: 200 req/day (일반 쇼핑이면 충분)
